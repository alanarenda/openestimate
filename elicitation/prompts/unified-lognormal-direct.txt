### If you choose Lognormal:

CRITICAL: μ (mu) and σ (sigma) are parameters in LOG-SPACE, not real-space!

Key relationships to real-space values:
- MEDIAN (real-space) = exp(μ)
- MEAN (real-space) = exp(μ + σ²/2)
- MODE (real-space) = exp(μ - σ²)

HOW TO SET μ:
1. First decide what you think the MEDIAN value should be (in the original units)
2. Then set μ = log(median)
3. Examples:
   - If median should be $30, then μ = log(30) ≈ 3.4
   - If median should be 100 employees, then μ = log(100) ≈ 4.6
   - If median should be $1000, then μ = log(1000) ≈ 6.9

HOW TO SET σ:
- σ controls the spread in log-space (typical values: 0.2 to 1.0)
- σ = 0.3 gives roughly a 95% credible interval of [exp(μ-0.6), exp(μ+0.6)]
- σ = 0.5 gives roughly a 95% credible interval of [exp(μ-1.0), exp(μ+1.0)]

COMMON MISTAKE TO AVOID:
❌ WRONG: Setting μ = 30 when you mean the value is $30
   (This gives median = exp(30) = $10 trillion!)
✓ CORRECT: Setting μ = log(30) ≈ 3.4 when you mean the value is $30
   (This gives median = exp(3.4) = $30)

ALWAYS VERIFY YOUR ANSWER:
- Calculate exp(μ) - does this match your expected median?
- Calculate exp(μ + σ²/2) - does this match your expected mean?
- If these are wildly different from what you expect, you've made an error!
